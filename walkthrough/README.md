# AvigilanceEnv Walkthrough

This folder is the operator guide for the Hugging Face Space, local validation, and submission flow.

Files in this folder:

- `space-frontend.md`: how to use the browser UI and what each panel verifies.
- `verification.md`: local commands for API, inference, Docker, and OpenEnv checks.
- `submission-checklist.md`: final pre-submit checklist aligned to the hackathon validator.

Recommended reading order:

1. Start with `submission-checklist.md`.
2. Use `space-frontend.md` when testing the Space manually.
3. Use `verification.md` when preparing a release or submission push.
