How To Verify This Space

1. Interactive checks

Use the console above to reset a task, inspect the observation payload, load an example action, and submit a step through the exact API that OpenEnv validators hit.

2. Submission validation

Run the repository validator in scripts/validate-submission.sh after the Space is deployed. It pings the Space, builds Docker, and calls openenv validate.

3. Full walkthrough docs

Repository docs live under the walkthrough/ folder and cover deployment, endpoint testing, inference expectations, and local verification commands.
